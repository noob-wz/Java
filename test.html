算法代码

function moveZeros(nums) {
let slow = 0; // 慢指针
for (let fast = 0; fast < nums.length; fast++) {
if (nums[fast] !== 0) {
nums[slow] = nums[fast];
slow++;
}
}
// 将剩余位置填充为零
for (let i = slow; i < nums.length; i++) {
nums[i] = 0;
}
}

算法步骤详解

初始状态

快指针和慢指针都位于数组起始位置(索引0)。数组包含零和非零元素。

步骤 1: 快指针遇到零

快指针遇到零元素，慢指针保持不动。快指针继续前进。

操作: 快指针移动到下一个位置，慢指针保持原位。

步骤 2: 快指针遇到非零元素

快指针遇到非零元素，将其值复制到慢指针位置，然后两个指针都向前移动。

操作: nums[slow] = nums[fast], 然后 slow++ 和 fast++。

步骤 3: 快指针再次遇到零

快指针遇到零元素，慢指针保持不动。快指针继续前进。

操作: 快指针移动到下一个位置，慢指针保持原位。

步骤 4: 快指针遇到非零元素

快指针遇到非零元素，将其值复制到慢指针位置，然后两个指针都向前移动。

操作: nums[slow] = nums[fast], 然后 slow++ 和 fast++。

步骤 5: 快指针遇到非零元素

快指针遇到非零元素，将其值复制到慢指针位置，然后两个指针都向前移动。

操作: nums[slow] = nums[fast], 然后 slow++ 和 fast++。

步骤 6: 填充剩余位置为零

快指针遍历完所有元素后，将慢指针之后的所有位置填充为零。

操作: 从 slow 到数组末尾，将所有元素设置为0。
