需求：使用 Java 开发零钱通项目，可以完成收益转账、消费、查看明细，推出系统等功能

```text
项目界面：       
--------------------------------- 零钱通菜单 ---------------------------------       
                            1. 零钱通明细        
                            2. 收益入账         
                            3. 消费       
                            4. 退出       

请选择（1-4）：           
--------------------------------- 零钱通明细 ---------------------------------           
收益入账        +100.0      2021-01-21 17:41        余额：100.0            
收益入账        +500.0      2021-01-21 19:40        余额：600.0            
真功夫          -20.0       2021-08-21 9:40         余额：580.0      
```
     


细节：     
（1）用户输入 4时，给出提示“你确定要退出吗 ? y/n", 必须输入正确的 y/n,否则循环输入指令，直到输入y或n        
（2）在收益入账或者消费时，判断金额是否合理，并给出相应提示      
（3）将面向过程的代码修改成面向对象的方法，编写 SmallChangeSysOOP.java 类，并用SmallChangeSysApp.java 完成测试
