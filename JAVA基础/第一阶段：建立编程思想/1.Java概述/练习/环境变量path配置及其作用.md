> **环境变量的作用是：操作系统用来查找可执行文件的路径列表**
>
> **配置过程（主要针对 windows 操作系统):**
> 第一步：**我的电脑 - 属性 - 高级系统配置 - 环境变量**
> 第二步：**创建 JAVA_HOME 变量，变量值为 JDK 的下载路径**
> 第三步：修改 path 的值。修改有两种方式，一种是用户配置，一种是系统配置，前者只能在当前用户下才能生效，后者是在任意用户环境下使用。具体配置是：**在 path 值的后面添加 %JAVA_HOME%\bin**
> 第四步：重新进入 DOS 系统，在任意目录下执行 java 或 javac 测试是否生效
