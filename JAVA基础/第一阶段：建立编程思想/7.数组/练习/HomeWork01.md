**题目 ———— 声明：int[] x,y[];以下选项允许通过编译的是（）：**
| 选项 | 您的理解 | 正确性 | 修正和解释 |
| :--- | :--- | :--- | :--- |
| **1. `x[0] = y;`** | 错误，`y` 是引用变量，`x[0]` 是具体的值，类型不匹配 | **✅ 正确** | `x[0]` 是 `int` (基本类型值)，`y` 是 `int[][]` (引用类型)，类型不兼容。 |
| **2. `y[0] = x;`** | 正确，`y[0]` 和 `x` 都是引用变量 | **✅ 正确** | `y[0]` 是 `int[]` 引用类型，`x` 也是 `int[]` 引用类型。一个 `int[]` 引用可以赋值给另一个 `int[]` 引用。 **注意：** 必须确保 `y` 已经被初始化，否则会抛出 `NPE`。 |
| **3. `y[0][0] = x;`** | 错误，`y[0][0]` 是具体的值，而 `x` 是引用变量，类型不匹配 | **✅ 正确** | `y[0][0]` 是 `int` (基本类型值)，`x` 是 `int[]` (引用类型)，类型不兼容。 |
| **4. `x[0][0] = y;`** | 错误，`x[0][0]` 未定义 | **✅ 正确** | `x` 是一维数组 `int[]`，访问 `x[0][0]` 会导致**编译错误**，因为它只有一维索引。 |
| **5. `y[0][0] = x[0];`** | 正确，相同类型变量之间的赋值 | **✅ 正确** | `y[0][0]` 是 `int` 类型值，`x[0]` 也是 `int` 类型值。相同基本类型之间的赋值是允许的。 **注意：** 必须确保 `x` 和 `y` 的所有维度都已初始化。 |
| **6. `x = y;`** | 正确，引用变量之间的赋值 | **❌ 错误** | `x` 是 `int[]` 类型引用，`y` 是 `int[][]` 类型引用。**类型不兼容**，会导致**编译错误**。 |

**说明：**

1. 由于二维数组的声明方式有三种：

   - (1) int[][] arr;
   - (2) int[] arr[];
   - (3) int arr[][];

   所以，**题目中的声明等价于：`int[] x; int[] y[];` 即声明一个一维数组和一个二维数组**

2. Java 不允许将一个数组引用赋值给一个比它维度少一级的数组引用
