1、当 `+` 左右两边的操作数**都是数值类型时**（如 float, int, double)，执行标准的加法运算，加号两边的操作数会先统一为其中 **精度最高的数据类型** ，然后再进行计算，**结果的类型也是精度更高的数据类型** 。其他算数运算符同样适用这个规则，而且运算的最小数据类型是`int`

2、当 `+` 左右两边的操作数**任一是 `String` 类型**时，它的功能会被重载为**字符串连接**。如果表达式中存在 String，那么 JVM 会将**其他非字符串类型的操作数（包括数字、布尔值、甚至对象）自动转换为它们的字符串表示形式**，然后将所有字符串连接起来。注意和`Char`类型参与运算的区别（即 **`''`和`""`的区别** ）

```java
String name = "Alice";
int age = 30;

// 1. 字符串与字符串连接
String greeting = "Hello, " + name; // "Hello, Alice"

// 2. 字符串与数字连接 (最常见用法)
String info = "Age: " + age + " years";
// 结果: "Age: 30 years" (数字 30 被自动转换为字符串 "30")

// 3. 字符串与布尔值连接
String status = "Login: " + true;
// 结果: "Login: true"
```
