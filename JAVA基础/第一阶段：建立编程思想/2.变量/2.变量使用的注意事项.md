1、变量表示内存中的一块存储空间
（不同的变量，类型不同，占用的空间大小不同。比如 int 占 4 个字节，double 占 8 个字节）

2、该空间有自己的名称[变量名]和类型[数据类型]

3、变量必须先声明，后使用

4、该空间的数据可以在同一类型范围不断修改。换句话说，变量值可以一直被修改

5、**变量的作用域直接控制了其内存的生命周期**

- **（1）变量在同一个作用域内不能重新声明**，Java 通常不允许在一个嵌套的作用域内重新声明一个与外部作用域同名的局部变量
  ```java
      public void testMethod() {
          int x = 10; // 外层作用域
          if (true) {
              // int x = 20; // ❌ 编译错误！C++会发生遮蔽，但Java不允许在嵌套块内重新声明 x。
          }
      }
  ```
- （2）**在互相独立的、不嵌套的作用域中，允许这种声明**。

  ```java
  public void testLoopScope() {
      // 动作 1: 进入第一个循环的作用域
      for (int i = 0; i < 5; i++) {
          System.out.println("第一次循环的 i: " + i);
      } // 动作 2: 离开第一个循环的作用域

      // 动作 3: 进入第二个循环的作用域
      for (int i = 0; i < 10; i++) {
          System.out.println("第二次循环的 i: " + i);
      } // 动作 4: 离开第二个循环的作用域

      // 补充说明：java 中，for 循环和 while 循环的“等价”仅限于控制流，但在作用域管理上，Java 的 for 语句拥有其独特的、强制性的语法规则，这种规则覆盖了通用的块级作用域规则。

       // for 循环
        // for (int i=1; i<5; i++) { ... }
        // i 的实际作用域： 仅限于 for 语句内部。相当于for语句外部有一个隐形的{}，如 {for (int i=1; i<5; i++) { ... }}

        // while 循环
        // int i = 1;
        // while (i < 5) { ... ; i++; }
        // i 的实际作用域： 覆盖整个外部方法块。
  }
  ```

| 动作       | 内存变化 (栈空间)                                                                                                       | 变量状态                                                          |
| :--------- | :---------------------------------------------------------------------------------------------------------------------- | :---------------------------------------------------------------- |
| **动作 1** | 栈帧为第一个 `i` **分配一块新的内存空间**（假设地址：`0xAAAA`）。                                                       | 变量名 `i` 映射到地址 `0xAAAA`。                                  |
| **动作 2** | 退出 `for` 循环块。系统将地址 `0xAAAA` 上的内存**释放并标记为可用**，意味着变量名与内存地址的**映射被销毁**。           | 变量 `i` (在 `0xAAAA` 上) **被销毁**，不再存在。                  |
| **动作 3** | 栈帧为第二个 `i` **再次分配一块新的内存空间**（系统很可能分配到 `0xAAAA`，因为它已空闲，或者分配到新的地址 `0xBBBB`）。 | **一个新的变量名 `i`** 映射到 `0xBBBB`。它与之前的 `i` 毫无关系。 |
| **动作 4** | 退出 `for` 循环块。系统将地址 `0xBBBB` 上的内存**释放**。                                                               | 变量 `i` (在 `0xBBBB` 上) **被销毁**。                            |

6、变量 = 数据类型 + 变量名 + 变量值
