实例变量和静态变量（也称为类变量）是定义 Java 类状态的两种主要方式，它们在存储位置、生命周期和访问方式上有本质的区别。

---

## 💻 实例变量 (Instance Variables)

实例变量是**对象级别的属性**。它们的值存储在类的每一个实例（对象）中，每个对象都有自己独立的一份变量副本。

### 1\. 概念

- **定义位置：** 声明在类中，但在任何方法、构造器或代码块之外。
- **存储位置：** 存储在 **堆 (Heap)** 内存中，作为对象的一部分。
- **生命周期：** 与它所属的**对象**生命周期相同。对象被创建时，实例变量被分配内存；对象被垃圾回收时，实例变量被销毁。

### 2\. 访问方式

- **访问：** 必须通过**对象实例**来访问（例如：`myObject.instanceVariable`）。
- **特点：** 每个对象拥有自己的值。改变一个对象的实例变量，**不影响**其他对象的同名实例变量。

### 3\. 声明示例

```java
public class Dog {
    // 实例变量
    String name;
    int age;

    // ...
}

// 使用示例：
Dog dog1 = new Dog();
dog1.name = "Max"; // dog1 有自己的 name

Dog dog2 = new Dog();
dog2.name = "Buddy"; // dog2 有自己的 name，且与 dog1 独立
```

---

## 🌐 静态变量 / 类变量 (Static Variables / Class Variables)

静态变量是**类级别的属性**。它们的值被该类的**所有实例**共享，是所有对象共同拥有的。

### 1\. 概念

- **定义标记：** 必须使用 `static` 关键字进行修饰。
- **存储位置：** 存储在 **方法区 (Method Area)**（或现代 JVM 中的元空间 Metaspace），在运行时只分配**一块**内存空间。
- **生命周期：** 与**类**的生命周期相同。类被加载到 JVM 时分配内存，JVM 关闭时才释放。

### 2\. 访问方式

- **访问：** 推荐通过**类名**来访问（例如：`Dog.COUNT`），也可以通过对象实例访问，但不推荐。
- **特点：** 所有对象共享**同一个**值。改变任何一个对象或通过类名改变静态变量的值，会**影响所有其他对象**。

### 3\. 声明示例

```java
public class Dog {
    // 静态变量 (类变量)，被所有 Dog 对象共享
    static int dogCount = 0;

    // 实例变量
    String name;

    public Dog(String name) {
        this.name = name;
        Dog.dogCount++; // 每创建一个对象，共享的计数器就增加
    }
}

// 使用示例：
Dog d1 = new Dog("Max");
Dog d2 = new Dog("Buddy");

// 访问：
System.out.println(Dog.dogCount); // 输出 2 (所有对象共享同一个计数)
```

## 📊 关键区别总结

| 特性         | 实例变量 (Instance Variables)  | 静态变量 (Static Variables)      |
| :----------- | :----------------------------- | :------------------------------- |
| **关键字**   | 无                             | **`static`**                     |
| **所属级别** | **对象**级别 (Object-level)    | **类**级别 (Class-level)         |
| **存储位置** | 堆 (Heap) 内存                 | 方法区 (Method Area) / 元空间    |
| **内存分配** | **每次**创建对象时分配**一份** | **仅在**类加载时分配**一份**     |
| **共享性**   | 独立，互不影响                 | **共享**，所有对象共用一个值     |
| **用途**     | 描述每个对象的**独有状态**     | 描述所有对象**共同的状态**或常量 |
