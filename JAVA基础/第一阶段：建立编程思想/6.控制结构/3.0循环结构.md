循环结构允许程序在满足某个条件时，重复执行某一段代码。这极大地提高了程序的效率，避免了重复编写代码。

## ➡️ 计数循环：`for` 循环

`for` 循环通常用于已知或可计算**循环次数**的情况。它清晰地定义了循环的三个关键要素：**初始化**、**条件判断**和**迭代更新**。

**结构四要素：**

1. **初始化 ：** 循环开始前只执行一次（如：$i = 0$）。初始化可以有多条，中间用 `,` 隔开。但是**所有初始化数据的类型要求一致**
2. **条件：** 每次循环开始前都要检查的条件（如：$i < 10$）。
3. **更新：** 每次循环体执行完毕后要执行的操作（如：$i++$）。更新语句也可以有多条
4. **循环体**

> 说明：
>
> 1. 由于 Java 中局部变量声明之后必须赋值，所以结构三要素中的**初始化**（主要是针对局部变量）肯定是**包含声明和赋值**的，可以写在 `for` 循环头部，也可以写在循环外部。
>
>    - 如果是 **实例变量** 或 **静态变量** ，“初始化”部分可以只赋值。
>
> 2. 当**变量初始化在外部** 或者 是**实例变量** 或 **静态变量**的赋值，`for` 循环和 `while` 循环是**可以互换**的。但**当在 `for` 循环内部初始化时，更新变量它的作用域仅针对 `for` 循环，此时和 `while` 结构的转换不等价**。
> 3. **`for` 循环的外部有一个隐藏的 `{}` 来限制了整个 `for` 循环的作用域**。

```java
    { // 隐藏括号，限定了for循环的作用域
        for (int i = 0; i < 10; i++) {
            // i 变量在这里是可见的
            System.out.println(i);
        }
    }

// 由于隐藏括号的原因，局部变量 i 只在循环内部分配内存空间，循环一结束，变量 i 就销毁了
// System.out.println(i); // ❌ 编译错误：找不到符号 i。
```

```java
// 死循环（无限循环）
for(;;)
```

## ➡️ 条件循环：`while` 循环

**逻辑流程：** **先判断条件，再执行循环体。**

#### 基本语法

```java
循环初始化;
while(布尔表达式) {
  循环体;
  循环变量更新;
}
```

## ➡️ 特殊条件循环：`do...while` 循环

**逻辑流程：** **先执行循环体，再判断条件。**

#### 基本语法

```java
循环初始化;

do {
  循环体;
  循环变量更新;
} while (布尔表达式);
```

**说明**

1. `do while` 是关键字
2. 最后有一个 **分号**
3. 和 `while` `for`一样都有循环结构体四要素，只是各要素位置不一样
4. while` 循环是**直到型循环**。它保证循环体**至少被执行一次**
