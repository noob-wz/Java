## 💡 Java 字符串的运算与比较

### 1\. 运算（部分支持）

Java 中的字符串 **不可以直接** 进行数学运算（如乘、除、减），但可以进行一种特殊的“运算”：**拼接 (Concatenation)**。

- **支持的运算：** **加法 (`+`)**
  在 Java 中，`+` 符号被重载用于字符串拼接。只要 `+` 号的任意一侧是 `String` 类型，另一侧的任何数据（包括基本类型和对象）都会被自动转换为 `String`，然后进行拼接。

  ```java
  String s1 = "Hello";
  String s2 = "World";
  String result = s1 + s2; // 结果是 "HelloWorld"

  int num = 5;
  String complex = "The number is " + num + 10; // 结果是 "The number is 510"
  ```

- **不支持的运算：** 减 (`-`)、乘 (`*`)、除 (`/`) 等数学运算。

### 2\. 比较（需要区分）

Java 中的字符串 **可以** 进行比较，但必须区分**引用比较**和**内容比较**。

#### ❌ 引用比较 (`==` 运算符)

使用 `==` 运算符比较两个字符串时，它比较的是这两个字符串变量存储的**内存地址（引用）**。

- 如果两个字符串变量指向内存中的**同一个 `String` 对象**，结果为 `true`。
- 如果两个字符串内容相同但存储在**不同的对象**中（即使内容完全一样），结果为 `false`。

因此，**不推荐**使用 `==` 来比较字符串的内容。

```java
String a = new String("test");
String b = new String("test");
System.out.println(a == b); // 结果通常是 false (两个不同的对象)
```

#### ✅ 内容比较 (`.equals()` 方法)

这是 Java 中比较字符串**内容是否相同**的**标准和正确**方式。`.equals()` 方法继承自 `Object` 类，并在 `String` 类中被重写，专门用于比较两个字符串对象序列中的字符是否完全一致。

```java
String a = new String("test");
String b = new String("test");
System.out.println(a.equals(b)); // 结果是 true (内容相同)
```

#### ✅ 字典顺序比较 (`.compareTo()` 方法)

如果你需要比较字符串的**大小顺序**（即字典顺序，谁在字母表中靠前），则使用 `.compareTo()` 或 `.compareToIgnoreCase()` 方法。它返回一个整数：

- `0`：内容相同。
- **负数：** 调用对象在参数对象之前。
- **正数：** 调用对象在参数对象之后。

### 总结表格

| 行为            | 运算符/方法    | 含义                               | 推荐用途                           |
| :-------------- | :------------- | :--------------------------------- | :--------------------------------- |
| **运算**        | `+`            | 字符串拼接（自动类型转换）         | 构造新字符串                       |
| **比较 (内容)** | `.equals()`    | 比较两个对象包含的字符序列是否一致 | **比较字符串内容**                 |
| **比较 (引用)** | `==`           | 比较两个变量是否指向同一个内存地址 | 仅用于基本类型或特殊对象（如枚举） |
| **比较 (大小)** | `.compareTo()` | 比较字符串的字典顺序               | 排序和查找                         |
