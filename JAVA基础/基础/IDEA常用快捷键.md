1. 删除当前行，自己配置 `ctrl + d`
2. 复制当前行，自己配置 `ctrl + alt + 向下光标`
3. 补全代码 `alt + /`
4. 添加注释或者取消注释 `ctrl + /`
5. 导入该行需要的类 `alt + enter` 
6. 快速格式化代码 `ctrl + shift + L`
7. 快速运行程序 `alt + R`
8. 生成构造器 `alt + N`
9. 查看一个类的层级关系 `ctrl + H`（配置“类型层级结构”）
10. 将光标放在一个方法名上，输入 `ctrl + B`，可以选择定位到对应类的方法实现部分
11. 自动的分配变量名，通过在后面 + `.val`
12. 自定义模版（可以高效提高开发速度）
    - `/v` 程序输出模版
    - `sout` 输出 `System.out.println()`
    - `fori` 循环模版
    - `now` 显示当前时间
13. 下方插入新行 `Shift + Enter`
